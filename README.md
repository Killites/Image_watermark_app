# 🖼️ Image Watermark App

## 📖 Overview
**This is a Python desktop application built using Tkinter**.  
It allows users to add **text or image watermarks** to pictures easily, with a simple and user-friendly interface.

## 🚀 Features
- **Add text watermark** to any image  
- **Add image watermark/logo** to images  
- **Preview images** before saving  
- **Save watermarked images** easily  

## 🛠️ Technologies Used
- **Python**  
- **Tkinter** (GUI)  
- **Pillow** (Python Imaging Library)  

## ▶️ How to Run
1. Clone the repository
2.Install required packages

-Bash
-pip install pillow
-Run the application

-Bash
-python main.py

-Use the GUI to select images and apply watermarks
